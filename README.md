# Healthy-Corgi
반려동물 건강 관리 다이어리, 헬시코기
<br><br>

## 🧨 설치 및 실행
- Installation

  ```
  $ cd corgi
  $ npm i
  $ npx react-native link
  ```
- Run

  ```
  $ npm start
  $ npm run android
  ```
